# 📊 Sales Performance Dashboard

An interactive Sales Performance Dashboard developed using Power BI to analyze sales trends, profitability, customer insights, and regional performance.

## 🚀 Features
- Total Sales KPI
- Total Profit KPI
- Total Orders
- Total Customers
- Monthly Sales Trend
- Sales by Category
- Sales by Region
- Sales by Segment
- Interactive Slicers (Date, Region, Category)

## 🛠 Tools Used
- Power BI
- Microsoft Excel
- Data Visualization
- DAX (Basic)
